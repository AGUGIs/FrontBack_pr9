# 🔐 Практическое занятие №9: Refresh-токены

Серверное приложение на Node.js с реализацией JWT-аутентификации, refresh-токенами и CRUD операциями для управления товарами.

---

## 📖 Описание

Приложение реализует:

- **Регистрацию пользователей** с хешированием паролей (bcrypt)
- **Вход в систему** с выдачей пары токенов (access + refresh)
- **Автоматическое обновление токенов** через refresh-токен
- **Ротацию токенов** (старый refresh-токен удаляется, создаётся новый)
- **Защищённые маршруты** с проверкой access-токена
- **CRUD операции** для управления товарами
- **Swagger документацию** API

---

## 🛠 Технологии

- **Node.js** — среда выполнения JavaScript
- **Express** — веб-фреймворк
- **bcrypt** — хеширование паролей
- **jsonwebtoken** — работа с JWT токенами
- **nanoid** — генерация уникальных ID
- **Swagger** — документация API

## 📥 Установка

1. Клонируйте репозиторий:
```bash
git clone <URL_РЕПОЗИТОРИЯ>
cd practical-09-refresh-tokens
```
2. Установите зависимости:
```bash
npm install
```

## 🚀 Запуск
```bash
# Режим разработки (с nodemon)
npm run dev

# Обычный запуск
npm start
```

Сервер будет доступен: http://localhost:3000
Swagger документация: http://localhost:3000/api-docs
